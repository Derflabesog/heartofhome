# Heart of Home Family Housing

This is the official website for **Heart of Home Family Housing**, providing affordable plot and plan construction services in Namibia.

## Project Structure

- **index.html**: The main landing page.
- **assets/**: Contains styles, scripts, and images.
  - `css/style.css`: Main stylesheet.
  - `js/main.js`: Main JavaScript file.
  - `images/`: Project images.

## Hosting

This site is designed to be hosted on GitHub Pages.
To view the live site, visit the GitHub Pages URL configured in the repository settings.
